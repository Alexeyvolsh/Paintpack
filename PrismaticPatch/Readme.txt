Патч, обеспечивающий поддержу ambient-light модуля Paintpack http://paintpack.ru/ в софте Prismatic https://code.google.com/p/lightpack/.

Реализована базовая версия DMX протокола для устройства с <= 10 светодиодными лентами и переход в режим бутлоадера.

Подключенное устройство с Vendor ID = 0x0E8F и Product ID = 0x0025 определяется автоматически.
